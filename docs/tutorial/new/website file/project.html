How to use Budgie?

Budgie is an open-source music programming kit. We provide the app, resources and curriculum to increase the access to programming education. You can start using Budgie in two steps:

1. Download the Budgie Music Programming app from:

2. Prepare the 3D blocks with a material of your choice, and use NFC stickers to define the models.

You can access the model files, code and guidelines from our

3. Start using Budgie with our tutorials!

2

Curriculum

We prepared a curriculum to introduce technology to young minds and develop their computational thinking skills.

The curriculum is intended to 9-12 age group, educators can adapt the curriculum for lower and upper age groups.

The curriculum includes activities from Ableton to teach about music and synth, and many other resources to teach the foundations making and coding.

3

Research and Development

Budgie is a tangible and low-cost programming kit for an inclusive programming education.

It is designed with the Tangible User Interface Design Guidelines.

We presented our work at TEI 2020 and awarded with the "Honorable Mention"!
